Системные требования

- Операционная система: Windows 7 / 8 / 10 / 11
- Установленная среда Visual Studio 2022 или новее
- .NET SDK (версия соответствует вашему проекту)
- Не менее 200 МБ свободного дискового пространства

Установка и запуск

Способ 1. Запуск через Visual Studio

1. Скачайте или склонируйте репозиторий на свой компьютер
2. Откройте папку с проектом
3. Найдите файл GameApp.sln и откройте его двойным щелчком мыши
4. Дождитесь загрузки решения в Visual Studio
5. Нажмите клавишу F5 для запуска с отладкой
6. Или нажмите Ctrl + F5 для запуска без отладки

Способ 2. Запуск через командную строку

1. Откройте командную строку или терминал
2. Перейдите в папку с проектом
3. Выполните команду dotnet run
4. Дождитесь компиляции и запуска приложения

Сборка проекта

Сборка через Visual Studio

1. Откройте решение GameApp.sln
2. В верхней панели выберите конфигурацию Debug (для отладки) или Release (для финальной версии)
3. Нажмите на меню Сборка -> Собрать решение
4. Или используйте сочетание клавиш Ctrl + Shift + B
5. Результат сборки появится в папке bin/Debug или bin/Release

Сборка через командную строку

Откройте терминал в папке с файлом GameApp.csproj и выполните:

dotnet build --configuration Release

Для публикации готового приложения:

dotnet publish --configuration Release --output ./publish

Готовый исполняемый файл будет находиться в папке publish.

Возможные проблемы и их решение

Проблема: Ошибка доступа к папке .vs

Решение: Добавьте папку .vs в файл .gitignore. Создайте файл .gitignore в корне проекта и пропишите в нём строку .vs/

Проблема: Не удаётся найти точку входа Program.cs

Решение: Убедитесь, что в проекте есть файл Program.cs с методом Main.

Проблема: Ошибка при запуске dotnet run

Решение: Установите .NET SDK с официального сайта Microsoft или откройте проект через Visual Studio.

Проблема: Сборка падает с ошибками

Решение: Проверьте, что все ссылки на библиотеки подключены правильно и версии .NET совпадают.
