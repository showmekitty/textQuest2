# Путь странника

Консольная игра в жанре roguelite. Выбор класса, бои, сундуки, босс.

## Управление
- Кнопки 1,2,3 — выбор действия
- Кнопка "Новая игра" — рестарт

## Классы
| Класс | Обычная атака | Усиленная атака |
|-------|---------------|-----------------|
| Мечник | 12 урона | 28 урона (КД 2) |
| Лучник | 9-13 урона + крит | 16-23 урона (КД 2) |
| Маг | 12 урона + поджог | заморозка (КД 4) |

## Игровой цикл
1. Разговор с NPC
2. Выбор оружия
3. Бой с орком (65 HP) → +25% HP
4. Сундук (бомба/зелье/пусто)
5. Бой с боссом (100 HP, оглушение)
6. Концовка (Рай или Ад)
